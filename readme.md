# Gitgit은 뭐하는 파일인가요?
Gitgit은 김현석이 대학교 수업시간 (python 프로그래밍)에서 배운 내용과 <br> *핫식스맨* 선생님께 배운 javascript에 대한 내용들을 제가 정리하거나 혼자서 이것저것 만져보며 끄적여보는 레포지토리입니다. <br>
__정말... 열심히 해야겠다는 생각이드네요..__

[] 2~3주차 내용들 (노트북 SSD고장으로 인해 다 날라감 슈밤)
